## 🚨 Gemini의 즉시 개선 권고사항

### 1순위: 프롬프트 개선 및 결과 포맷팅 (1.5시간) ⚠️

**문제점**:
- AI가 잘못된 복지 정보를 제공할 수 있음
- 결과가 일관되지 않을 수 있음
- 정보를 받은 후 "다음에 뭘 해야 하지?" 막막함

**해결책**:

#### A. 프롬프트에 정보 출처 명시
```python
prompt = """
당신은 대한민민국 복지 전문가입니다.
**중요**: 보건복지부 '복지로' 사이트나 각 지자체 공식 홈페이지의 2025년 최신 데이터를 기반으로 답변해주세요.
만약 확실하지 않은 정보라면 '가까운 주민센터에 문의가 필요합니다'라고 명시해주세요.

사용자 상황:
{user_input}

아래 형식으로 답변해주세요:
"""
```

#### B. 'Next Action' 추가
```json
{
  "benefit_name": "기초연금",
  "description": "만 65세 이상, 소득 하위 70%",
  "amount": "월 최대 32만원",
  "next_action": "신분증을 가지고 가까운 주민센터에 방문하여 신청하세요. (문의: 국번없이 129)"
}
```

#### C. JSON 포맷으로 결과 통일
```python
response = model.generate_content([
    """다음 형식의 JSON으로 답변해주세요:
    {
        "benefits": [
            {
                "name": "복지 혜택명",
                "target": "대상",
                "amount": "금액",
                "description": "설명",
                "next_action": "다음 할 일 (구체적으로)",
                "contact": "문의처"
            }
        ]
    }
    """,
    audio_file
])
```

**예상 소요 시간**: 1.5시간
**우선순위**: 🔥 최우선 (지금 바로!)
