## 🎬 영상 제작 시 사용법

### 1️⃣ 나레이션 (설명 음성)

**용도**: 영상 전체를 설명하는 내레이터 목소리

**선택**:
- `narration_old_male/` - 남자 나이든 목소리 (신뢰감)
- `narration_old_female/` - 여자 나이든 목소리 (친근함) ⭐ 추천

**타임라인 배치**:
```
00:00-00:30  01_intro.mp3
00:30-01:00  02_service.mp3
01:00-01:10  03_scenario1_intro.mp3
01:10-01:50  04_scenario1_result.mp3
01:50-02:00  05_scenario2_intro.mp3
02:00-02:40  06_scenario2_result.mp3
02:40-02:50  07_scenario3_intro.mp3
02:50-03:30  08_scenario3_result.mp3
03:30-03:50  09_tech_stack.mp3
03:50-04:10  10_differentiation.mp3
04:10-04:30  11_outro.mp3
```

---

### 2️⃣ 모의 사용자 음성 (시연용)

**용도**: 실제 어르신이 앱을 사용하는 것처럼 보이게 함

**파일**:
- `character_grandma.mp3` - 72세 독거 할머니 (시나리오 1)
- `character_grandpa.mp3` - 68세 저소득 할아버지 (시나리오 2)
- `character_senior.mp3` - 70세 건강 문제 어르신 (시나리오 3)

**사용 예시**:

#### 시나리오 1: 실시간 녹음 시연
```
1. 나레이션: 03_scenario1_intro.mp3
   "72세 독거 할머니의 사례를 보여드리겠습니다"

2. 화면: 녹음 버튼 클릭

3. 모의 음성: character_grandma.mp3 재생
   "저는 72살이고 혼자 살고 있어요. 다리가 아파서 거동이 불편합니다"
   (실제로는 직접 말하거나, 음성 파일 재생)

4. 화면: AI 분석 → 결과 표시

5. 나레이션: 04_scenario1_result.mp3
   "AI가 어르신의 상황을 분석하여..."
```

#### 시나리오 2: 텍스트 입력 시연
```
1. 나레이션: 05_scenario2_intro.mp3
   "68세 저소득 어르신의 사례입니다"

2. 화면: 텍스트 타이핑
   "68살이고 소득이 적어서 힘들고 일자리를 찾고 싶어요"
   (character_grandpa.mp3 내용을 타이핑)

3. 화면: 복지 혜택 찾기 클릭 → 결과

4. 나레이션: 06_scenario2_result.mp3
```

#### 시나리오 3: 음성 파일 업로드 시연
```
1. 나레이션: 07_scenario3_intro.mp3

2. 화면: 음성 파일 업로드 버튼 클릭
   → character_senior.mp3 파일 선택
   (또는 미리 준비한 MP3 파일)

3. 화면: 결과 표시

4. 나레이션: 08_scenario3_result.mp3
```

## 💡 Pro Tips

### Tip 1: 나레이션 목소리 선택
```
✅ narration_old_female/ 추천
이유:
- 어르신들에게 더 친근함
- "복지 도우미" 느낌
- 따뜻하고 부드러움

또는:
✅ narration_old_male/
이유:
- 신뢰감, 권위
- 공식적인 안내 느낌
```

### Tip 2: 모의 음성 활용 방법

**방법 A: 직접 녹음** (가장 자연스러움)
```
장점: 진정성, 자연스러움
단점: 녹음 품질 관리 필요
→ 시간 있으면 이 방법 추천!
```

**방법 B: AI 음성 파일 사용** (편리함)
```
장점: 빠름, 품질 일정
단점: 약간 기계적
→ 시간 없으면 character_*.mp3 사용
```

**방법 C: 하이브리드** (추천!)
```
시나리오 1: 직접 녹음 (가장 중요한 부분)
시나리오 2: 타이핑 (AI 음성 불필요)
시나리오 3: character_senior.mp3 사용
```

### Tip 3: 영상 편집 시 주의사항

**볼륨 레벨**:
```
나레이션: -3dB ~ 0dB (메인 음성)
모의 사용자: -6dB ~ -3dB (약간 낮게)
BGM: -18dB ~ -12dB (배경으로만)
```

**페이드 인/아웃**:
```
각 오디오 클립:
- 시작: 0.2초 페이드 인
- 끝: 0.2초 페이드 아웃
→ 매끄러운 전환
```

**오디오 간격**:
```
나레이션 → 모의 음성: 0.5초 간격
모의 음성 → 나레이션: 0.3초 간격
```
