## 💡 iMovie 사용법

### 1. 프로젝트 생성
- 해상도: 1920x1080
- 프레임레이트: 30fps

### 2. 파일 Import
```
1. iMovie 열기
2. 미디어 Import (⌘I)
3. "audio/male 복사본/" 폴더 전체 선택
4. Import
```

### 3. 타임라인 배치
```
- 위 타임라인 순서대로 드래그앤드롭
- 슬라이드 이미지도 Import
- iPhone 녹화본도 Import
- 순서대로 배치
```

### 4. 오디오 레벨 조정
```
- 나레이션: 0dB (기본)
- BGM (선택): -15dB (매우 낮게)
```

### 5. 렌더링
```
- 파일 → 공유 → 파일 (⌘E)
- 해상도: 1080p
- 품질: 높음
- 형식: MP4
```
